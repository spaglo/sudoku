# Sudoku — ЛР 1 (парная разработка)

Реализация игры Судоку на C# (.NET 8, WinForms) с разделением на расчётную часть и графический интерфейс.

## Команда

| Студент | Часть | Папка |
|---|---|---|
| **A** | Игровой движок и тесты | `Sudoku.Engine/`, `Sudoku.Engine.Tests/` |
| **B** | WinForms-интерфейс | `Sudoku.UI/` |

## Архитектура

```
Sudoku.UI  ──→  Sudoku.Engine          (UI знает про Engine)
                       ↑
              Sudoku.Engine.Tests       (тесты знают про Engine)
```

Engine — Class Library, не зависит от WinForms. Может быть переиспользована в любом другом UI (WPF, ASP.NET, консоль).

## Правила игры

Классическое Судоку 9×9 с тремя элементами реального времени:
- **Таймер** — секундомер от начала партии.
- **Жизни** (3 шт.) — за каждый неверный ввод списывается одна. При 0 жизней — поражение.
- **Подсказки** (3 шт.) — кнопка «Hint» раскрывает одну случайную пустую клетку.

Сложность определяет число пустых клеток в начале партии (Easy: ~40 пустых, Expert: ~55).

## Управление

| Действие | Клавиатура | Мышь |
|---|---|---|
| Выбрать клетку | стрелки ←↑→↓ | клик левой кнопкой |
| Поставить цифру | 1 … 9 | — |
| Стереть цифру | 0, Backspace, Delete | — |
| Новая игра | Меню → Новая игра | кнопка |
| Подсказка | Ctrl+H | кнопка |
| Перезапустить паззл | Ctrl+R | кнопка |

## Сборка и запуск

```pwsh
dotnet build Sudoku.sln
dotnet test               # тесты движка
dotnet run --project Sudoku.UI
```

Требуется .NET 8 SDK. WinForms-часть работает только на Windows.
